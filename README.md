# ESP32音乐播放器（带NFC控制）

## 概述

这是一个基于ESP32的音乐播放器项目，具有NFC控制功能。该系统使用状态机架构来管理不同的操作模式，包括启动、播放器和NFC管理模式。它具有SD卡用于音乐存储，使用LVGL图形库的UI显示，以及用于用户交互的NFC功能。

## 功能

- **状态机架构**: 使用不同应用状态实现关注点的清晰分离
- **音乐播放**: 具有音量控制的音频播放系统
- **NFC集成**: NFC标签读取和管理
- **SD卡存储**: 来自SD卡的音乐文件管理
- **LVGL GUI**: 使用LVGL图形库的触摸屏UI
- **ESP32-S3平台**: 专为ESP32-S3开发板优化

## 硬件组件

- **ESP32-S3 DevKitC-1**: 主微控制器板
- **SD卡模块**: 用于音乐文件存储
- **NFC读取器(RC522)**: 用于NFC标签交互
- **显示屏**: 使用TFT_eSPI库的TFT显示屏
- **音频输出**: I2S音频接口

## 软件架构

应用程序使用状态机模式，包含以下状态：

### 应用状态

1. **STATE_BOOT** - 初始化阶段
   - SPI总线初始化
   - 存储(SD卡)初始化
   - UI系统初始化
   - 转换到播放器状态

2. **STATE_PLAYER** - 音乐播放模式
   - 音乐播放器的UI显示
   - 音频播放控制
   - SD卡上的文件浏览

3. **STATE_NFC_ADMIN** - NFC管理模式
   - NFC标签读取和配置
   - 管理功能

4. **STATE_RADIO** - 收音机模式(计划功能)

### 关键组件

- **UI系统**: 使用LVGL图形库构建触摸界面
- **存储系统**: 用于音乐文件的SD卡集成
- **音频系统**: 高级音频播放接口
- **NFC系统**: 标签读取和管理功能
- **板级抽象**: 引脚和SPI的硬件抽象层

## 依赖项

- **LVGL**: v8.3.11 - UI图形库
- **TFT_eSPI**: v2.5.43 - 显示驱动
- **SD**: Arduino SD库 - SD卡访问
- **MFRC522**: v1.4.10 - NFC/RFID库

## 构建和运行

1. 安装PlatformIO
2. 在PlatformIO中打开项目
3. 构建并上传到ESP32-S3 DevKitC-1板

项目配置为：
- ESP32-S3 DevKitC-1板
- Arduino框架
- 115200波特率串口监视器
- 核心调试级别3
